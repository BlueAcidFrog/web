For developers

By developers

Integrate immediately

Our API is a real time-saver. It consists of only one method: ory().authorize. You will call this method from your log in endpoint, to let us know who is trying to get authorization.

ory().authorize(request, {
  user: "jane.doe@example.org"
})

Don't worry about access control

You can trust us to only let users past who present proper authorization. Gone are the days where you have to deal with ACL, RBAC and other funkyness. Simply build your app.

For easy processing, you receive user information via HTTP headers.

const handler = (request, response) => {
  const user = req.get('X-FIREWALL-SUBJECT');
  // ...
}

Works everywhere

The protocols we use have libraries in any major programming language. Authorizing users on the client side is a breeze. This particular example uses passportjs.

passport.authenticate('oauth2');
